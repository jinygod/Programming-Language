변수(Variable)

(변수)
1. 변수란, 하나의 값을 저장할 수 있는 메모리 공간
2. 변하는 수를 담을 수 있는 그릇
3. 메모리 번지에 대한 이름(별칭)

(변수 작성 규칙)
1. 영문자, 숫자, 특수문자($, _)의 조합
2. 특수문자는 달러($)와 언더스코어(_)만 쓸 수 있다.
3. 첫번째 문자는 숫자가 나와서는 안된다. : 9abc(x)
4. 길이의 제한이 없다. (너무 길면 가독성이 떨어짐, 적당한 길이)
5. 자바 예약어(Reserved Word)는 사용할 수 없다.
	-컴파일 오류가 나온다.
6. 대소문자 구별 : abc, Abc는 서로 다른 변수
7. 변수의 예 : abc, a123, $abc, _abc, $hello, _hello, $124

(변수의 선언)
1. 자료형 변수이름;
2. 자료형 변수이름 = 초기값;
3. 예제: 정수형(integer) 변수를 선언
	int x;
	int y = 10;
	int z = -9;
4. 자바10 이후 버전(지역변수만 가능)
	var 변수이름 = 초기값;
	
(데이터 타입)
1. 정수(byte, char, short, int, long) : 0, 양수, 음수
2. 실수(float, double) : 부동소수점, 지수
3. 논리(boolean) : 참과 거짓
4. 사용자(class, enum) : 정수, 실수, 논리의 복합적으로 조합
5. 원시타입(Primitive Type): 기본타입(정수, 실수, 논리)

(할당 연산자: =)
1. 변수 = 값 (값 -> 변수)
2. 데이터를 변수에 대입
3. 기존의 변수에 있던 값은 사라지고 대입되는 값이 저장
3. 예 : 변수 x를 선언하고 초기값으로 10을 할당한 후에 99를 넣으면
		기존에 있던 값인 10은 사라지고 99가 변수 x에 저장된다.
	int x = 10; // x <- 10
	x = 99;     // x <- 99
	
	자바버전 10 이후
	var x = 10;
	var s = "abc";
	
(리터럴: Literal)
1. 정수리터럴, 실수 리터럴, 논리 리터럴, 문자 리터럴, 문자열 리터럴
2. 예: 10, -5, 0.14, 'a', "abc", true, false
3. 문자리터럴: 'a', '한', '\uAC00'
4. 문자열리터럴: "문자열", "abc", "welcome"

(상수: constant)
1. 변하지 않는 수
2. 변하지 않는 문자 및 문자열
3. 한 번 지정되면 값을 바꿀 수 없는 변수(variable)

(원시타입: primitive type)
1. 논리형: boolean(1byte) 
2. 정수형: byte(1byte) < short(2byte), char(2byte) < int(4byte)
3. 실수형: float(4byte) < double(8byte)

(원시타이의 값의 범위)
1. 논리(boolean) : true, false(0, 1)
2. 바이트(byte) : -128 ~ 127
3. 문자(char) : 0 ~ 65535
4. 쇼트(short) : -32768 ~ 32767
5. 정수(int) : -2147483648 ~ 2147483647
6. 실수(float) : 1.4E-45 ~ 3.402E38
7. 실수(double) : 4.9E-324 ~ 1.797E308

(바이트 타입) -128 ~ 127
최상위비트(부호비트): MSB(Most Significant Bit), 0:양수, 1:음수
+127 : 0111 1111
  +2 : 0000 0010
  +1 : 0000 0001
   0 : 0000 0000
  -1 : 1111 1111
  -2 : 1111 1110
-128 : 1000 0000

--------------------------
+1 : 0001          (2의보수 = 1의보수 + 1)
-1 : 1111
--------------------------
+2 : 0010
   : 1101 (1보수)
   +    1 (2보수)
-2 : 1110
--------------------------
+10: 1010
-10: 0110
