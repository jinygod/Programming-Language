상속(Inheritance)

1. 객체지향 관점에서 상속?
- 부모 클래스에 속성과 기능을 상속 받는 자식 클래스에서 내 것처럼 사용
- 부모 클래스의 기능: 속성(멤버필드), 기능(메소드)을 자식 클래스에서 
  자신의 것처럼 접근 할 수 있다.
  
2. 동물(Animal)
- 공통적인 특징: 살아있다. 호흡한다. 움직인다. 먹는다.
- 특징을 속성과 행위(기능)으로 분류
  > 속성: 상태나 특징, 살아있다, 호흡한다.
  > 기능: 먹는다, 움직인다.
  
3. 새(Bird)
  > 속성: 상태나 특징, 살아있다, 호흡한다.
  > 기능: 먹는다, 움직인다.(걷거나 난다)

4. 호랑이(Tiger)
  > 속성: 상태나 특징, 살아있다, 호흡한다.
  > 기능: 먹는다, 움직인다.(걷거나 뛴다)

5. 물고기(Fish)
  > 속성: 상태나 특징, 살아있다, 호흡한다.
  > 기능: 먹는다, 움직인다.(헤엄친다)
