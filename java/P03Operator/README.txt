[연산자]

1. 연산자의 종류
	- 대입연산자
	- 부호연산자
    - 산술연산자
    - 비교연산자
    - 논리연산자
    - 증감연산자
    - 비트연산자
    - 조건연산자

2. 피연산자(Operand)
	- 연산되는 데이터
	
3. 연산식(Expression)
	- 연산자와 피연산자를 이용하여 연산 과정을 기술
	
4. 연산의 방향
	- 좌측 -> 우측
	
5. 대입연산자(Assignment Operator)
	- 변수 = 변수 또는 식
	- 오른쪽의 처리 결과를 왼쪽의 변수에 대입
	- 대입이 되는 왼쪽의 변수는 오른쪽의 값으로 대치됨
	- 예: x = 10;
	
6. 부호연산자(Sign Operator)
	- 단항 연산자
	- 양수(+), 음수(-)
	- 양수는 부호 생략 가능
	
7. 산술연산자(Arithmetic Operator)
	- 처리 결과는 숫자
	- +, -, *, /, %
	
8. 비교연산자(관계 연산자)
	- 처리 결과는 논리형(boolean)
	- == : 같다
	- != : 같지 않다
	- >  : 크다
	- >= : 크거나 같다
	- <  : 작다
	- <= : 작거나 같다
	
9. 논리연산자
	- && : 논리곱(and)
	- || : 논리합(or)
	- !  : 부정(not)
	
10. 증감연산자
	- ++ : 1씩 증가
	- -- : 1씩 감소
	
11. 비트연산자
	- &, |, ^, ~, <<, >>, >>>
	- & : 비트 곱(and)
	- | : 비트 합(or)
	- ^ : 배타적 비트 합(XOR)
	- ~ : 비트 반전
	- <<: 비트 왼쪽으로 이동
	- >>: 비트 오른쪽으로 이동(왼쪽 비트에 부호가 채워짐)
	- >>> : 비트 오른쪽으로 이동(왼쪽 비트에 0이 채워짐)
	
12. 연산자 우선순위
	- 일차식    : 왼쪽 -> 오른쪽
	- 단항(*)   : 오른쪽 -> 왼쪽, !, ++, --, +, -
	- 산술 	    : 왼쪽 -> 오른쪽
	- 비트 이동 : 왼쪽 -> 오른쪽
	- 관계      : 왼쪽 -> 오른쪽
	- 비트 곱   : 왼쪽 -> 오른쪽
	- 비트 합   : 왼쪽 -> 오른쪽
	- 논리 곱   : 왼쪽 -> 오른쪽
	- 논리 합   : 왼쪽 -> 오른쪽
	- 조건      : 왼쪽 -> 오른쪽
	- 대입(*)   : 오른쪽 -> 왼쪽,
				 복합대입 연산자(=, +=, -=, *=, %=, /=, <<=, >>=, >>>=, &=, |=, ^=)
				 
13. 조건연산자(삼항연산자)
	- 변수 = 조건식 ? 참 : 거짓;
	- 조건식이 참이면 ? 뒤의 참의 값이 변수에 대입
	- 조건식이 거짓이면 ? 뒤의 거짓의 값이 변수에 대입
